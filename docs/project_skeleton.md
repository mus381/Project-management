# Daily Settlements Project Skeleton

## Purpose
Learn, practice, and simulate a fintech daily settlements workflow.

## Folder Overview
- **data/** → raw CSV/JSON files of transactions
- **scripts/** → Python scripts for processing and analysis
- **docs/** → notes, references
- **kanban.md** → task tracking board

## Day 1 Focus
1. Understand what daily settlements are
2. Setup project folder
